
## 🚀 Vercel 배포 방법

1. 이 폴더를 GitHub 저장소로 Push
2. [vercel.com](https://vercel.com) → New Project → GitHub 연결
3. Framework Preset: **Other** 선택
4. Root Directory: `/` (기본값)
5. Deploy 클릭
